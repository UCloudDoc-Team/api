



# 混合云 API 简介

欢迎您选择 混合云（UHybridV3），API 适合通过编写程序，自动化管理和编排云上资源的场景。

## 使用限制

如果您的资源用量超过了配额限制，可以联系客户经理，或提交工单申请。

## 阅读指南
